1. Simple Neural Network (Single Input):
This example shows how a basic neural network learns from one input. The model first predicts a value of 3.0, which is lower than the target 5.0, resulting in error and loss. Through backpropagation, it calculates gradients and updates the weights using gradient descent. After updating, the prediction improves to 3.78, showing that the model is learning by reducing the error.

2. Backpropagation Using Matrices:
This example demonstrates learning in a larger neural network using matrices. The initial prediction is 1.15, far from the target 4.0, leading to high loss. Using matrix-based backpropagation, the model computes gradients and updates both weight matrices. After the update, the prediction improves to 2.33, indicating that the network is successfully learning and moving closer to the target.
